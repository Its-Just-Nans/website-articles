## Extract the APK from your device

</summary>

### With `APK Extractor`

- install an `APK Extractor` app from the Store
- launch the app
- select the app you want to extract
- the app will create an `.apk` file

### With `adb` <Details>https://developer.android.com/tools/adb</Details>

- install `adb` on your computer
- get the full app name with `adb shell pm list packages | grep APP_NAME`
- get the apk path with `adb shell pm path FULL_APP_NAME`
- pull the apk with `adb pull APK_PATH`

</details>

<details>
<summary>

## `apktools` <Details>https://github.com/iBotPeaches/Apktool</Details>
`apktool` can extract the `.apk` file into a directory with the resources and the classes

</summary>


- Follow the docs on https://apktool.org/docs/install
- download the latest release
- run `java -jar apktool.jar d APP_NAME.apk`

</details>

{/* ------------------------- */}

<details>
<summary>

## `dex2jar` <Details>https://github.com/pxb1988/dex2jar</Details>

`dex2jar` can transform the `.apk` into a `.jar` file viewable by `jd-gui`

</summary>


- https://github.com/pxb1988/dex2jar/releases
- download the latest release
- extract the zip file
- run `d2j-dex2jar.sh APP_NAME.apk`

</details>

{/* ------------------------- */}

<details>
<summary>

## `jd-gui` <Details>https://github.com/java-decompiler/jd-gui</Details>

`jd-gui` can view the `.jar` file generated by `dex2jar`

</summary>

- https://github.com/java-decompiler/jd-gui/releases
- download the latest release
- run `java -jar jd-gui.jar APP_NAME-dex2jar.jar`
- you can now see classes

</details>

{/* ------------------------- */}

<details>
<summary>

## `jadx` <Details>https://github.com/skylot/jadx</Details>

`jadx` can decompile the `.apk` file and view the source code

</summary>


- https://github.com/skylot/jadx/releases
- download the latest release
- run `jadx -d out_folder APP_NAME.apk`
- you can now see the source code
- run `jadx-gui` (and open the APK inside ) to see the source code in a GUI


</details>

## Get signatures of the APK

```bash
apksigner verify --print-certs example.apk
```
